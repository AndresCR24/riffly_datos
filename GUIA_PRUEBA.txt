🎯 MÉTODO 1: SCRIPT AUTOMÁTICO (MÁS FÁCIL)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

En la terminal, ejecuta:

Este script automáticamente:
✓ Verifica que cx_Oracle esté instalado

✓ Prueba la conexión a Oracle
✓ Verifica todos los archivos
✓ Te muestra todas las URLs disponibles
✓ Inicia el servidor Django

🎯 MÉTODO 2: PASO A PASO MANUAL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PASO 1: Configurar Contraseña
──────────────────────────────────────────────────────────────

Abre el archivo: db_connection.py

Busca la línea 13 que dice:
   'password': 'tu_contraseña',

Cámbiala por tu contraseña real de Oracle:
   'password': 'MI_CONTRASEÑA_REAL',

Guarda el archivo (Cmd+S en PyCharm)


PASO 2: Probar la Conexión
──────────────────────────────────────────────────────────────

En la terminal ejecuta:

   python db_connection.py

✅ RESULTADO ESPERADO:
   🔍 Probando conexión a Oracle (Riffly Music Platform)...
      Usuario: riffly_m
      DSN: localhost:1521/XE
   --------------------------------------------------
   ✅ Conexión exitosa!

   📊 Estadísticas de la base de datos:
      • USERS: X registros
      • SONGS: X registros
      • GENRES: X registros
      • PLAYLISTS: X registros
      • COMMENTS: X registros
      • LIKES: X registros
      • PLAYS: X registros
      • FOLLOWS: X registros

❌ SI HAY ERROR:
   - Verifica que Oracle esté corriendo
   - Verifica que la contraseña sea correcta
   - Verifica que el usuario 'riffly_m' exista
   - Verifica que las tablas estén creadas


PASO 3: Iniciar el Servidor
──────────────────────────────────────────────────────────────

En la terminal ejecuta:

   python manage.py runserver

Verás algo como:
   Starting development server at http://127.0.0.1:8000/
   Quit the server with CONTROL-C.

⚠️ SOLUCIÓN DE PROBLEMAS COMUNES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROBLEMA: "Error de conexión a Oracle"
SOLUCIÓN:
   1. Verifica que Oracle esté corriendo:
      - Abre PyCharm
      - Ve a la pestaña de Database
      - Verifica que la conexión "riffly_m" esté activa

   2. Verifica la contraseña en db_connection.py

   3. Intenta conectarte desde PyCharm SQL Console

PROBLEMA: "Página en blanco o error 500"
SOLUCIÓN:
   1. Revisa la consola del servidor (terminal)
   2. Verifica que las tablas existan en Oracle
   3. Ejecuta: python db_connection.py para verificar

PROBLEMA: "No se muestran datos"
SOLUCIÓN:
   Las tablas están vacías. Necesitas:
   1. Crear usuarios primero
   2. Crear géneros
   3. Luego crear canciones
   4. Finalmente crear playlists

PROBLEMA: "cx_Oracle not found"
SOLUCIÓN:
   pip install cx_Oracle

PROBLEMA: "Template not found"
SOLUCIÓN:
   Verifica que existan estas carpetas:
   - templates/usuarios/
   - templates/canciones/
   - templates/generos/
   - templates/playlists/
   - templates/comentarios/
